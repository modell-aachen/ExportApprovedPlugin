---++ Plugin Preferences

There are no global preferences for this plugin, but individual topics
affected by it may use preferences to control how the topic is treated:

   * EXPORT_AS_PDF: set to non-empty to export the topic; otherwise it is ignored
   * PDF_LANDSCAPE: set to 1 to use landscape orientation when generating the PDF
   * PDF_PAPERSIZE: set to A3 to use DIN A3 instead of A4
